The drawing we generated is editable in two different ways.

Placing the cursor at the drawing (just after or just before, the drawing is then surrounded by a thin cyan frame) and pressing brings back the yellow edit window, where the code can be changed and re-executed into a new drawing.

It is also possible to edit the drawing with the interactive (point and click) facilities. In this case too it is always possible to return to the text-editing mode of the environment by pressing with the cursor at the drawing, but if one does that, the interactive modifications are lost, i.e. one gets back to the code one had typed into the environment.
