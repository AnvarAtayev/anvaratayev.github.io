Here is a detailed guide to build GNU TeXmacs using CMake and Homebrew.

Homebrew is a package manger for macOS. Typically, we install Homebrew first, and then use the command line to install build tools and dependencies to build TeXmacs.

Xcode Command Line Tools provide the C/C++ compiler. It is required. TeXmacs can not be built without a C/C++ compiler. And TeXmacs is designed to be customizable using Scheme. If you want to be a power user of TeXmacs but not want to compile TeXmacs, please browse the built-in documents under .
